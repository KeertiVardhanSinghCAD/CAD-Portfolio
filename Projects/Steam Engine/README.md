# Steam Engine

## Overview

A fully parametric steam engine assembly designed in FreeCAD. This project demonstrates mechanical assembly design, part modeling, technical documentation, and motion visualization.

## Highlights

- 30+ individual components
- Fully assembled in FreeCAD
- Technical drawing package
- STEP export
- Assembly animation

## Skills Demonstrated

- Parametric Part Design
- Mechanical Assembly
- Technical Drawings (TechDraw)
- Design Documentation
- Assembly Constraints

## Deliverables
