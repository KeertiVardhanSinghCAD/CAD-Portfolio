# Hand Crank Gearbox

## Overview

A manually operated gearbox modeled in FreeCAD to demonstrate gear assemblies, motion transfer, and parametric mechanical design.

## Highlights

- Functional gear train
- Fully assembled mechanism
- Technical drawing package
- STEP export

## Skills Demonstrated

- Gear Design
- Assembly Modeling
- Parametric CAD
- Technical Drawings

## Deliverables
